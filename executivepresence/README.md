# Executive Presence Staging Mockup

This folder is a local staging workspace for Executive Presence site updates.

The `mockup/` directory contains static HTML captures of the live Squarespace pages. These files are intended for review, copy/layout experiments, and side-by-side comparison before approved changes are manually applied in Squarespace.

## Pages

- `mockup/index.html` - Home
- `mockup/solutions.html` - Solutions
- `mockup/assessments.html` - Assessments
- `mockup/about.html` - About
- `mockup/contact.html` - Contact

## Workflow

1. Use these files to propose and review changes locally.
2. Keep live Squarespace untouched until changes are approved.
3. Apply approved copy/layout changes to hidden or duplicated Squarespace pages first.
4. Promote the staged Squarespace page to live only after review.

## Notes

- The mockup uses live Squarespace/CDN assets so it visually tracks the current site closely.
- Navigation for captured pages has been rewritten to local `.html` files.
- Store, cart, book, privacy, and terms links still point to the live site because they were not captured for this mockup.
